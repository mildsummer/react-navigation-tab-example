#### Install

```
npm install
```
or
```
yarn
```

### Usage

#### Run the app

```
npm run start
```
or
```
yarn run start
```

#### Develop the app for iOS

```
npm run ios
```
or
```
yarn run ios
```

#### Develop the app for Android

```
npm run android
```
or
```
yarn run android
```
